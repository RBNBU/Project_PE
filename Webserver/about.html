Detailed Project Overview

CarRadio Monitor is an advanced engineering project that merges traditional Amateur Radio (APRS) protocols with modern geospatial web technologies. By capturing digital packets transmitted over VHF frequencies, we transform raw radio telemetry into an interactive, real-time tracking experience.

This system is designed to provide high-reliability monitoring of vehicle dynamics including velocity, altitude, and precise positioning while simultaneously integrating live atmospheric data. It serves as a proof-of-concept for bridging hardware-level C programming with cloud-based data visualization.

How it Works

1
Mobile Data Acquisition

A Raspberry Pi 2B inside the vehicle interfaces with a GPS module to collect real-time coordinates, altitude, and velocity.

2
APRS Transmission

The system encodes the gathered telemetry into standardized AX.25 APRS packets for radio transmission.

3
RF Reception

A dedicated ground station equipped with a high-gain antenna monitors VHF frequencies to capture the incoming packets.

4
C-Based Backend Processing

A custom C program parses the raw radio data, filtering for our specific callsign to ensure data integrity.

5
API Enrichment

The system automatically queries a Weather API using the car's current coordinates to fetch local atmospheric conditions.

6
Data Centralization

Processed telemetry and environmental data are pushed to a central database for live access and historical logging.

7
Web Dashboard

Frontend fetches database updates and visualizes real-time info.
